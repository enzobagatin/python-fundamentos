# Python Fundamentos

Repositório criado para consolidar os fundamentos da linguagem Python com foco em Ciência de Dados.

## Conteúdos
- Variáveis e tipos de dados
- Listas
- Estruturas de repetição (for)
- Condicionais (if, elif, else)
- Funções

## Objetivo
Construir uma base sólida em Python para análises de dados, automação e projetos futuros.
